Game Play Instructions:

  • Use the On/Off button to turn the game on and off.
  • The Start & Strict buttons are disabled while the game is off.
  • Use the Start button to start or stop a game.
  • Use the Strict button to enable or disable strict game play. In this mode the game restarts at a new random series of button presses.
  • The number of turns the player has successfully repeated are show in the Score.
  • The player wins after successfully repeating 20 steps.
